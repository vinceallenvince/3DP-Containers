##Jar Base

Sliced using (Simplify3D)[http://www.simplify3d.com] for Makerbot Replicator 2.

Layers 0 - 10mm:
  * Layer height: 0.3mm
  * Perimeter shells: 3
  * Bottom solid layers: 3
  * Top solid layers: 0
  * Infill: 10%
  * Fill pattern: Rectilinear
  * Outline overlap: 50%
  * Allow gap fill when necessary.
  * Coast distance: 0.2mm
  * Wipe nozzle: 5mm

Layers 10mm - 16mm:
  * Layer height: 0.2mm
  * Perimeter shells: 6
  * Bottom solid layers: 3
  * Top solid layers: 3
  * Infill: 100%
  * Fill pattern: Rectilinear
  * Outline overlap: 50%
  * Allow gap fill when necessary.
  * Coast distance: 0.2mm
  * Wipe nozzle: 5mm

Layers 16mm - 17mm:
  * Layer height: 0.3mm
  * Perimeter shells: 2
  * Bottom solid layers: 0
  * Top solid layers: 3
  * Infill: 10%
  * Fill pattern: Concentric
  * Outline overlap: 50%
  * Allow gap fill when necessary.
  * Coast distance: 0.2mm
  * Wipe nozzle: 5mm

To try to reduce blobs:
Coasting 1.6mm (stops pushing filament "x" distance from the end of a line)
Wipe nozzle 5mm
